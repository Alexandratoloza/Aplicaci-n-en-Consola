Aplicación de Consola en JavaScript

Proyecto del Módulo 4 - DESARROLLO DE APLICACIONES FRONT-END TRAINEE

Descripción

Este proyecto permite practicar los fundamentos de JavaScript:

  • Operaciones matemáticas básicas
  • Condicionales y bucles
  • Funciones y modularización
  • Arreglos y objetos

Abre la consola del navegador (F12 → Consola) para interactuar con la aplicación.

Instrucciones

  1. Ingresa tu nombre cuando se solicite.
  2. Proporciona dos números para realizar operaciones.
  3. Filtra frutas por letra.
  4. Observa cómo se usan objetos y arreglos.
